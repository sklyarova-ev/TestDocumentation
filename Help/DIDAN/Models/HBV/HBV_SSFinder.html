HBV_SSFinder

PURPOSE

function ss=HBV_SSFinder(coef)

SYNOPSIS

function ss=HBV_SSFinder(coef)

DESCRIPTION 

 function ss=HBV_SSFinder(coef)
 с помощью программы math.exe системы Wolfram Mathematica
 находит значения стационарных состояний модели Марчука-Петрова для 
 фиксированного набора параметров.  

 Функцию можно рассматривать как пример функции, которую 
 НЕОБХОДИМО задать пользователю для анализа его модели с помощью функции 
 SS_analysis. 

 EXTERNAL FUNCTIONS: math 

 INPUTS: 
          coef : упорядоченный в заданном пользователем порядке массив 
                 значений параметров при которых будут найдены стационарные
                 состояния. 
 OUTPUTS: 
           ss : найденные стационарные состояния, записанные в двумерный 
                массив размерностью NxM, где N - число стационарных
                состояний, а М - размерность системы уравнений (в случае
                модели Марчука-Петрова М = 10).
